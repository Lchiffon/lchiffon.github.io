第一部分 简介

介绍

Vue.js 是一套轻量级js框架, 可以用于构建自己相关的html服务等.
Vue.js在设计之初的目标是通过尽可能简单的API去实现响应数据绑定和组合的视图套件

特性:

  • 单文件组件支持
  • Vue生态库
  • Github
  • 在线编辑器
  • 中文VUE教程网站

引用:

vue与vue-cli:
vue-cli是vue的一个命令行工具, 在熟悉vue之后会进行学习.
